# 🔥 Karma5 — Guía de despliegue en 3 pasos

## Opción A: Subir a Vercel (GRATIS, 5 minutos)

### Paso 1 — Sube el proyecto a GitHub
1. Ve a https://github.com/new
2. Crea un repositorio nuevo llamado `karma5` (dejarlo en público)
3. Descarga GitHub Desktop: https://desktop.github.com
4. Arrastra la carpeta `karma5` al GitHub Desktop
5. Pulsa "Commit to main" → "Push to origin"

### Paso 2 — Conecta con Vercel
1. Ve a https://vercel.com
2. Regístrate con tu cuenta de GitHub (gratis)
3. Pulsa "Add New Project"
4. Selecciona el repositorio `karma5`
5. Deja todo por defecto → pulsa **Deploy**

### Paso 3 — Tu URL está lista
En 2 minutos tendrás una URL tipo:
`https://karma5-tuusuario.vercel.app`

---

## Instalar en iPhone

1. Abre **Safari** en tu iPhone (obligatorio Safari, no Chrome)
2. Ve a tu URL de Vercel
3. Pulsa el botón de compartir (cuadrado con flecha ↑ abajo)
4. Baja y pulsa **"Añadir a pantalla de inicio"**
5. Pulsa **"Añadir"**

¡Ya tienes el icono de Karma5 en tu iPhone como una app! 🎉

---

## Opción B: Ejecutar en local

```bash
npm install
npm run dev
```

Abre http://localhost:3000 en tu navegador.
